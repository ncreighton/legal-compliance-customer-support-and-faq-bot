# Legal & Compliance Customer Support and FAQ Bot

24/7 automated responses, ticket routing, and knowledge base search for Legal & Compliance businesses. Support coverage without hiring more staff.

## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
